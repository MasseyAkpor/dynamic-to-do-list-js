# Dynamic To-Do List Application

A modern, responsive To-Do List web application with local storage persistence.

## Features

- ✅ **Add new tasks** with Enter key or button click
- ✅ **Mark tasks as complete** by clicking on them
- ✅ **Remove tasks** with dedicated remove buttons
- ✅ **Persistent storage** using localStorage (tasks survive page refresh)
- ✅ **Task statistics** showing total and completed tasks
- ✅ **Responsive design** that works on all devices
- ✅ **Smooth animations** and modern UI

## How to Use

1. Type a task in the input field
2. Press Enter or click "Add Task" to add it
3. Click on a task to mark it as completed/incomplete
4. Click the "Remove" button to delete a task
5. All tasks are automatically saved to your browser
